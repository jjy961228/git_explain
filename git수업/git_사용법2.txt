6. reset을 했을경우 다시 돌아가는 방법
즉, "inital commit!" 으로 reset했을때 다시 "5 test"로 돌아가는 방법
	
	git log로 commit 상태 확인 
	git log

	inital commit! 으로 돌아가기(reset하기)
	git reset hard --3cad59d4193c448c4278f76259a3e2e320d466ba
	-->폴더를 확인해보면 아무것도 없는상태가 되었다~
		하지만 , git홈페이지에서는 최신버전으로 되어있다		
		이때는, git레포지토리 모두지우고 클론을 다시하던지
		git홈페이지에서 second_project로 들어가 commit 코드를 복사해오는 방법이 있다 
	git 홈페이지에서 commit 코드 복사해오기
	레포지토리 들어가서 commit 시점보기 -> 커밋코드 확인 
	->23094f50a0f65a72fb19cec00dc5dfa3727873dd 복사해오기

	->C:\shine\gitProjects\second_Project>git reset --hard 23094f50a0f65a72fb19cec00dc5dfa3727873dd  이렇게해주면 폴더에서도  git 레포지토리와 똑같이 된다 ~
